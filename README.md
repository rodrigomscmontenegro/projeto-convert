# 💸 Convert - Conversor de Moedas

O **Convert** é uma aplicação prática para conversão de moedas (Dólar, Euro e Libra) para Real Brasileiro (BRL). Este projeto foi desenvolvido como um exercício guiado durante os estudos na **Rocketseat**, focando intensamente na manipulação do DOM e lógica de programação com JavaScript.

---

## 🎯 Foco do Projeto

Como o HTML e o CSS foram fornecidos como base estrutural pelo curso, o desenvolvimento foi centrado na implementação de toda a inteligência da aplicação via **JavaScript Vanilla**.

### Principais conceitos aplicados:

- **Manipulação do DOM**: Captura de eventos de formulário e alteração dinâmica de conteúdos na página.
- **Validação com Regex**: Uso de expressões regulares (`/\D+/g`) para garantir que o input de valor aceite apenas números, limpando caracteres inválidos instantaneamente.
- **Estruturas de Controle**: Uso de `switch` para gerenciar as diferentes taxas de câmbio.
- **Internacionalização (Intl)**: Uso do método `toLocaleString` para formatar os valores numéricos no padrão monetário brasileiro (R$).
- **Tratamento de Erros**: Implementação de blocos `try/catch` para capturar exceções e fornecer feedback ao usuário.

---

## 🛠️ Tecnologias Utilizadas

- **HTML5** (Estrutura)
- **CSS3** (Estilização e Layout)
- **JavaScript (ES6+)** (Lógica e Interatividade)
- **Google Fonts** (Fontes Roboto, Inter e IBM Plex Mono)

---

## 🚀 Como executar

1. Clone ou baixe este repositório.
2. Abra o arquivo `index.html` em seu navegador.
3. Insira o valor, escolha a moeda desejada e clique em **"Converter em reais"**.
4. O resultado aparecerá dinamicamente no rodapé da página.
